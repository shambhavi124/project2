ðŸ“Œ Working Principle

The ultrasonic sensors continuously measure the distance between the sensor and any object in the parking slot. Based on the distance value, the system identifies if a vehicle is present. This data is transmitted through the ESP8266 microcontroller to the Blynk cloud, where users can view the live status of each slot.
